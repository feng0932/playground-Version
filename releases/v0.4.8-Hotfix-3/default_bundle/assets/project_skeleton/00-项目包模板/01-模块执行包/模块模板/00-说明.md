# 模块包说明

本页用于说明当前模块包的正式入口与边界。

## 入口

- 模块 PRD 真源：`01-PRD/01-模块PRD.md`
- 模块流程图与人类确认：`01-PRD/02-模块流程图与人类确认.md`
- 模块工程交接入口：`02-工程交接/README.md`

## 边界

- 模块 `PRD` 是产品真源
- `02-模块流程图与人类确认.md` 是模块级正式下游文件，用于承载页面流转、点击流、规则卡核对与人类确认回链
- 原型前总控负责检查 `02-模块流程图与人类确认.md` 是否已落盘并回链真源；产品审查负责原型前质量兜底
- `02-模块流程图与人类确认.md` 只做核对与回链，不重写 `PRD` 正文规则
- 模块 `02-工程交接/README.md` 是工程交接正式入口
- 阅读层不是事实源，只能帮助人理解，不能反向替代模块真源

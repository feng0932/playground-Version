# 模块工程交接正式目录索引

本目录是模块级工程交接的一等入口，用于列出当前有效真源、历史保留文件和替换关系。

## 当前有效真源

- 当前有效真源应按角色或主题分组列出
- 必要时补充 `linked_prd_source`，但只允许回链到模块 `PRD` 真源

## 历史保留

- 历史保留文件必须写清 `retired-truth marker`
- 若已有替换文件，必须写清 replacement 路径

## 目录规则

- 本 README 是正式目录索引，不再使用“未完成工程交接”的 starter 口径
- 若目录内已有真实工程交接材料，必须在这里显式暴露当前有效真源

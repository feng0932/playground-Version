# 项目包索引

本页是项目级人类入口，帮助读者快速找到项目基线、模块入口和工程交接入口。

## 稳定入口

- 项目级基线：`00-项目包/`
- 项目级合规专题入口：`00-项目包/04-开发前合规附录.md`
- 项目级原型底座入口：`00-项目包/05-项目级原型与高保真标准.md`
- 项目级工程交接入口：`00-项目包/02-工程交接/README.md`
- 模块总入口：`01-模块执行包/00-模块索引.md`
- PRD阅读入口：`80-完整提交包/01-合并PRD/00-PRD索引.md`
- 对外提交阅读入口：`80-完整提交包/00-提交说明.md`

## 使用边界

- `00-项目包/` 与 `01-模块执行包/` 承担真源职责
- `80-完整提交包/` 只承担阅读层与展示层职责
- 工程交接是一等入口，不得藏在过程文件或口头说明里

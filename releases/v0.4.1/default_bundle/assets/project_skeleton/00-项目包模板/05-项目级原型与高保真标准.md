## 6. 联动节点入口要求

- `prototype preflight / 原型前准备` 未完成时，不得直接外推为底座可用。
- 页面合同、状态清单或结构化契约字段未冻结时，不得把缺口转嫁给原型底座阶段。
- 项目级原型 authority 与模块级 `PRD / 4.2A / 4.3 / 4.4` 发生冲突时，必须先回产品真源修正，再继续原型执行。
